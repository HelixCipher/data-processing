# Chicago Crime Data Analysis

### Overview

This project analyzes crime data in Chicago from 2008-2012, along with selected census data. The goal is to explore correlations between crime rates and socioeconomic factors.

### Datasets

- Chicago Crime Data: Crimes 2001 to Present

- Chicago Census Data: Selected Socioeconomic Indicators

### Features

* Data preprocessing and cleaning.

* Merging census and crime data.

* Exploratory data analysis and visualization.

* Efficient data handling using vectorized operations.

### Author

This project was developed as part of a data analysis study.
